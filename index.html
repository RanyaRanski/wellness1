Політика конфіденційності

Aloe Wellness Coach є wellness / lifestyle застосунком для підтримки щоденних звичок. Він не є медичним сервісом, не призначений для діагностики, лікування чи профілактики захворювань і не надає медичних порад.

1. Які дані ми можемо збирати

  • облікові дані, якщо ви створюєте акаунт електронною поштою;
  • профіль wellness-цілей, тривалість плану, reminder settings і обрані вподобання;
  • записи про воду, сон, вагу, настрій, нотатки та виконання плану;
  • технічні події на кшталт outbound clicks, crash/debug logs і базових analytics hooks;
  • локальні дані пристрою, потрібні для нагадувань і безпечного збереження сесії.

2. Як використовуються дані

  • для створення персоналізованої wellness-рутини;
  • для відображення трекерів і прогресу в застосунку;
  • для роботи нагадувань і збереження ваших налаштувань;
  • для покращення стабільності сервісу, безпеки та якості роботи застосунку.

3. Інфраструктурні сервіси

Для технічної роботи сервісу можуть використовуватися Supabase, Firebase Cloud Messaging, захищене локальне сховище на пристрої та інші стандартні інфраструктурні провайдери. Ми не продаємо ваші персональні дані.

4. Зовнішні product links

Деякі кнопки відкривають зовнішні сторінки продуктів або магазину. Після переходу на сторонній сайт застосовуються його власні privacy policy, cookies та checkout rules. Рекомендуємо ознайомитися з ними окремо.

5. Ваш контроль над даними

Ви можете змінювати свої записи, редагувати профіль, виходити з акаунта, використовувати гостьовий режим або звернутися на підтримку щодо privacy-запитів. Якщо для production release у вас доступне створення акаунта, перед App Store submission також перевірте наявність in-app account deletion.

6. Контакти

Для privacy та support-запитів: support@aloewellnesscoach.app
